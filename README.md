# Distributed Time Series Management System 📊

**Grade: 17.7/20** ⭐

Project for the **Distributed Systems** course
Bachelor's Degree in Computer Engineering
University of Minho
Academic Year 2025/2026

This project consists of a client-server system for managing time series of sales events, with support for aggregations, real-time notifications and disk persistence.

---

## 🛠️ Usage

To start the server:
```bash
$ ./run-server.bat    # Windows
$ ./run-server.sh     # Linux/Mac
```

To start a client:
```bash
$ ./run-client.bat    # Windows
$ ./run-client.sh     # Linux/Mac
```

To run automated tests:
```bash
$ ./run-tests.bat     # Windows
$ ./run-tests.sh      # Linux/Mac
```

To run performance tests:
```bash
$ ./run-perf-tests.bat    # Windows
$ ./run-perf-tests.sh     # Linux/Mac
```
